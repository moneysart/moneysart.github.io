ที่มาที่ไป และความตั้งใจ ในการจัดทำ website นี้.

... เริ่มจาก เรามองหา เครื่องมือ ช่วยวางแผนเกษียณ ที่ใช้ง่าย
ที่ผ่านมาต้องทำ Excel ผูกสูตร ซึ่งเสียเวลา ถ้าคนที่ไม่เชี่ยวชาญแล้วก็จะลำบากเพิ่มไปอีก.

... จึงตัดสินใจสร้าง เครื่องมือวางแผนการเงิน ในรูปแบบ web-based tools
เพื่อให้ ทุกคนสามารถเข้าถึง เครื่องมือวางแผนการเงิน ที่ใช้ง่าย เข้าใจง่าย และเหมาะกับคนไทย.

... หลังจากคิดไอเดีย 2-3 อย่างที่จะทำแล้ว ก็รวบรวม ทำเป็น website ขึ้น
นอกจากนี้ admin ยังเป็นผู้แนะนำการลงทุน (Independent Financial Advisor กับ Finnomena)
จึงได้จัดทำ สรุปข้อมูลข่าวสารการลงทุนรายสัปดาห์ ไว้ในที่นี้ด้วย
เพื่อประโยชน์ของผู้เยี่ยมชม ที่เป็นผู้สนใจการวางแผนทางการเงิน การลงทุน.

... และท่านที่สนใจ ยังสามารถติดตามเราได้ทาง Facebook

... ท่านใด มีข้อแนะนำ/อยากให้เราเพิ่มเติมเครื่องมืออะไร เชิญส่งข้อความมาได้ที่ Facebook Page ครับ.

... โฆษณา ท่านใดลงทุนหรือสนใจลงทุนกับ Finnomena
และอยากให้ admin เป็นผู้ดูแล ก็สามารถติดต่อมาได้เลยครับ

-ขอให้มีความสุขกับการลงทุนครับ
