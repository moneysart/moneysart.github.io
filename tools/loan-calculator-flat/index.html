โปรแกรมคำนวณสินเชื่อ/เงินกู้ ดอกเบี้ยแบบคงที่

สำหรับ สินเชื่อเช่าซื้อรถยนต์, รถจักรยานยนต์, และอื่นๆ
แนะนำใช้หน้าจอแนวนอน สำหรับมือถือจอเล็ก Share

ดอกเบี้ยเงินกู้แบบคงที่ หรือ ดอกเบี้ยเงินกู้แบบเงินต้นคงที่ นั้น ภาษาอังกฤษ เรียกว่า fixed rate หรือ flat rate.

การคิดดอกเบี้ยแบบนี้ มีการใช้สำหรับ สินเชื่อเช่าซื้อรถยนต์ หรือรถจักรยานยนต์.

โดยวิธีการคิดนั้น จะทำการคำนวณดอกเบี้ยที่ต้องชำระทั้งหมดจากเงินต้นทั้งก้อน และนำมาแบ่งเป็นภาระดอกเบี้ยตามจำนวนงวดผ่อนชำระ
ดังนั้น จึงเป็นที่มาของชื่อ ว่า เป็นการคิดดอกเบี้ย แบบเงินต้นคงที่.

เช่น หากพิจารณาตาม chart ที่เรานำเสนอในหน้านี้ ก็จะเห็นได้ว่า
การเพิ่มของดอกเบี้ยที่ผู้กู้ต้องจ่ายในแต่ละงวดนั้น มีจำนวนเท่ากันตลอดอายุระยะเวลากู้ยืม.

การกู้ยืมสินเชื่อประเภทนี้ ไม่มีความจำเป็นต้องจ่ายโปะ เนื่องจากไม่เกิดประโยชน์ในการลดภาระหนี้.

หวังว่าจะช่วยให้คุณมีสุขภาพการเงินที่ดีขึ้นครับ.

ALL TOOLS วางแผนเกษียณ คำนวณ DCA สินเชื่อ ลดต้นลดดอก สินเชื่อ ดอกเบี้ยคงที่ ดอกเบี้ยทบต้น

เราใช้สมมติฐานว่า งวดชำระนับเป็นรายเดือน และแต่ละงวดการชำระมีจำนวนวันเท่ากัน
โดยในความเป็นจริง ผู้ให้สินเชื่อบางแห่งอาจใช้การคำนวณแบบนับจำนวนวันจริงในแต่ละงวด
ผลลัพธ์ที่เราแสดงจึงอาจแตกต่างจากการคำนวณแบบละเอียดอยู่เล็กน้อย

ดอกเบี้ยรวม = เงินต้น * ดอกเบี้ยรายปี * (จำนวนเดือนที่ผ่อนชำระ / 12)

ภาระการชำระต่อเดือน = (เงินต้น + ดอกเบี้ยรวม ) / จำนวนเดือนที่ผ่อนชำระ

ยอดชำระดอกเบี้ยสำหรับแต่ละเดือน = ดอกเบี้ยรวม / จำนวนเดือนที่ผ่อนชำระ

ยอดชำระเงินต้นสำหรับแต่ละเดือน = ภาระการชำระต่อเดือน - ยอดชำระดอกเบี้ยสำหรับแต่ละเดือน

เงินต้นคงเหลือ = เงินต้นที่เหลือจากงวดก่อนหน้า - ยอดชำระเงินต้นสำหรับแต่ละเดือน
